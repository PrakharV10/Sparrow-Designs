Font Family
Sparrow uses Roboto, as its default font, which is a free font available at Google Fonts. 
                        
    /* By Default imported in Sparrow */

    @import url('https://fonts.googleapis.com/css2?family=Roboto:wght@100;300;400;500;700;900&display=swap');
