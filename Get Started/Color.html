Sparrow's Color Scheme
Sparrow comes with pre-defined colors, which are consistent in the overall design. The colors used are chosen on a generic basis, which can be accessed by pre-defined classes.
Default Palette
Below is the color palette, used by Sparrow, listed with the hex-codes.
