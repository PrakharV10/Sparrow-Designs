Badge
Badges are small circles, positioned either at top-right or bottom-right of the parent component. Badge can be used to display numbers, online / offline status, depending on where they are used.
Profile Badge
Profile Badge is used to show the current status of a user.
It can be used with 3 variants : online | offline | dnd
                        
                    <-- Using the 'badge' class on parent and variant class on child -->

                    <div class="badge" >
                        <img class="avatar" src="https://url-of-the-image" alt="avatar-1">
                        <div class="online"></div>
                    </div>

                    <div class="badge" >
                        <img class="avatar" src="https://url-of-the-image" alt="avatar-1">
                        <div class="offline"></div>
                    </div>

                    <div class="badge" >
                        <img class="avatar" src="/assets/svg/female profile.svg">
                        <div class="dnd"></div>
                    </div>
Number Badge
Number Badge is used to show the no. of items present in the parent component.
10
15
                        
                    <-- Using the 'badge' class on parent and 'number' class on child -->

                    <div class="badge" >
                        <img class="avatar" src="/assets/svg/cart.svg" alt="avatar-1">
                        <div class="number">10</div>
                    </div>

                    <div class="badge" >
                        <img class="avatar" src="/assets/svg/mail.svg" alt="avatar-1">
                        <div class="number dnd">15</div>
                    </div>
