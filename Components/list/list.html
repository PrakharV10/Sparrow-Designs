List
Lists are continuous, vertical indexes of text or images. They are composed of items containing primary and supplemental actions, which are represented by icons and text.
Simple List Groups
    Categories
  • Home
  • Smartphones
  • Earphones
  • Watches
                        
                <-- Using 'list' class name -->
                
                <ul class="list">
                    <div class="list-title">Categories</div>
                    <li>Home</li>
                    <li>Smartphones</li>
                    <li>Earphones</li>
                    <li>Watches</li>
                </ul>
Nested List Groups
    Categories

    • Realme

                <-- Using 'list' class name, and 'collapsable' for creating nested list -->
                
                <ul class="list">
                    <div class="list-title">Categories</div>
                    <li>Home</li>
                    <li class="collapsable">
                        <div>Smartphones</div>
                        <ul>
                            <li>Apple</li>
                            <li>Realme</li>
                            <li>Samsung</li>
                        </ul>
                    </li>
                    <li>Earphones</li>
                    <li>Watches</li>
                </ul>
