Avatar
Avatars are user profile pictures, usually found in circular shapes. Avatars can be used for 'container' objects — projects, spaces, repositories, etcetera — to give them a recognisable visual identity.
Profile Avatar
Profile Avatars can be created using the class 'avatar', inside the img tag. 
                        
                    <-- Using the 'avatar' class name -->

                    <img class="avatar" src="https://url-of-the-image" alt="avatar-1">
                    <img class="avatar" src="/assets/svg/female profile.svg">
Sizes Avatar
Size of the avatars can be changed by adding sm | md | lg class to the existing avatar class. 
                        
                    <-- Using the sm | md | lg class names -->

                    <img class="avatar sm" src="https://url-of-the-image" alt="avatar-1">
                    <img class="avatar lg" src="https://url-of-the-image" alt="avatar-1">
                    <img class="avatar md" src="/assets/svg/female profile.svg">
Square Avatar
Adding a class 'square' to existing class, will generate a square shaped avatar.
Note : Square class wouldn't work if the image provided is svg. 
                        
                    <-- Using the square class -->

                    <img class="avatar square sm" src="https://url-of-the-image" alt="avatar-1">
                    <img class="avatar square md" src="https://url-of-the-image" alt="avatar-1">
                    <img class="avatar square lg" src="https://url-of-the-image" alt="avatar-1">
